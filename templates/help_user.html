  • Login

    In order to log in, follow the following steps:

    • Enter the login url on browser
    • Enter email and password
    • Enter matatu service name
    • Click login

  • Register

    In order to register, follow the following steps:

    • Click register
    • Enter all the required information in the required format on the registration form
    • Follow the instructions on the screen

  • Book

    In order to book a ticket, follow the following steps:

    • Log in to the traveller front end
    • Click book on the tab
    • Select departure location
    • Select destination location
    • Select matatu service
    • Select destination location if a matatu is available for that destination
    • Select seat number
    • Enter mpesa pin on your mobile phone

  • History

    In order to view your ticket history, click history on the tab

  • Logout

    In order to log out, click log out on the tab
